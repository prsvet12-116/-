# Лабиринт Тьмы

Интерактивная хоррор-игра с лабиринтами, сгенерированными по алгоритму Прима.

## Описание игры
Игрок должен пройти 5 уровней лабиринта, собирая кристаллы, находя ключи и избегая духов.

## Особенности
- 5 уровней сложности
- Генерация лабиринтов по алгоритму Прима
- Атмосферный хоррор-геймплей
- Адаптивный дизайн (ПК и мобильные)
- 6 уровней сложности

## Управление
- **ПК**: WASD или стрелки
- **Мобильные**: свайпы по экрану

## Цель
Собрать все кристаллы, найти ключ и выйти через дверь, избегая духов.
